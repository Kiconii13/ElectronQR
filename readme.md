# ElectronQR

A simple QR code generator and scanner built with Electron.

## Features

- Generate QR codes from text or URLs
- Customize QR foreground, background, and size
- Add a logo to QR codes
- Scan QR codes from images
- Keep a history of generated/scanned codes

## Installation

```bash
git clone <repo-url>
cd ElectronQR
npm install
npm start
